ViewEk is a GUI tool which allows the user to view the contents of any E kernel. It has capabilities similar to NAIF's command-line tool called 'inspekt'.

The file called 'ViewEk.form' was generated by the Netbeans (aka Forte For Java) IDE when ViewEk was built using the Netbeans GUI editor. This file will be needed to edit ViewEk using Netbeans/Forte For Java GUI editor in the futre.
